# Giita

## Installation

```
go get github.com/kaneshin/giita/cmd/giita
```

_~/.giita_

```
{
  "team": "YOUR TEAM NAME",
  "token": "YOUR ACCESS TOKEN"
}
```

## Usage

### post

```
$ giita post sample.md -tags=foo,bar -title="This is title"
```

### team

```
$ giita team
```
